## To-Do сервис (React)

### Локальный запуск

```bash
npm install
npm run dev
```

### Тесты

```bash
npm test
```

### Сборка

```bash
npm run build
```

### Деплой на GitHub Pages

1. Запушьте проект в GitHub (ветка `main`).
2. В репозитории откройте Settings → Pages.
3. В поле Build and deployment выберите Source: GitHub Actions.
4. Сделайте push в `main` (или запустите workflow вручную).

Workflow: [.github/workflows/deploy.yml](file:///Users/lunis/WebstormProjects/to-do-react/.github/workflows/deploy.yml)
